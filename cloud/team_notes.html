🛰️ GooTeX Team Collaboration Portal

💡 Quick Info for the Team:

  • Shared Environment: Only one person needs to run this server for the entire team to compile.
  • Persistence: If you are the host, do not close this tab or your laptop. The server dies when your connection breaks.
  • Automatic Logic: This environment pulls the latest compiler engine from GitHub on startup. No manual updates required.
🔧 Troubleshooting & Settings (Click to Expand)

• Server Offline? If a co-author sees a "Connection Error" in the Google Doc, click Runtime > Run all here.

• Timezone: To change PDF timestamps, edit the value in Cell (1) and re-run the server.

• Logs: Compiler logs are saved to your /Compiled folder in Drive for deep debugging.
